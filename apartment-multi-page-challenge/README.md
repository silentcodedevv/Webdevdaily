In this project, you'll build a Apartment multi-page website!

This project involves building a multi-page apartment website. It is a great way to test your HTML and CSS skills while gaining experience structuring and organizing a larger-scale website.

## Tasks

1. Build out this multi-page website and get it looking as close to the design as possible.
2. Create responsive layout for mobile and desktop based on provided design.
3. Implement hover states for interactive elements.

Use any tools you'd like to help you complete the project. If you've got something you'd like to practice, feel free to try it!

# Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 414px
- Desktop: 1440px
