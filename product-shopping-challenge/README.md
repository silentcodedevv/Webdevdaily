# Product List With Cart

## Welcome! 👋

Practice your HTML, CSS, and JavaScript by building a functional Shopping Cart page. Let users add items to their cart, view item details, update quantities, and see the total cost. Style it your way and make it your own!

This project presents a fantastic opportunity to work with a frontend JavaScript framework like Vue or React. However, feel free to use any tools or technologies you're comfortable with to complete the project. If there's something you're eager to practice or explore, go ahead and give it a try.

## Tasks

# Social Links Profile

1. Build out this product list with cart and get it looking as close to the design as possible.
2. View appropriate layout depending on the device screen size.
3. Add items to the cart.
4. Update the quantity of an item once it has been added to the cart.
5. Ensure items can be removed from the cart.
6. Add hover states for interactive elements.
7. Bonus: Save cart items to local storage so cart persists state on refresh.
